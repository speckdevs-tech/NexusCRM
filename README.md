# NexusCRM – Outreach Lead Management System

A premium LocalStorage-powered CRM built for freelancers, agencies, sales teams, and outreach professionals who need to manage leads efficiently without relying on expensive SaaS platforms.

## Overview

NexusCRM helps you:

* Add and manage leads
* Track outreach progress
* Monitor follow-ups
* Record conversations and notes
* Manage sales pipelines
* Analyze conversion performance
* Export and backup data

All data is stored directly in the browser using LocalStorage.

No backend required.
